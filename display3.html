Abbreviation

It is used to mark up an abbreviation or acronym:

US got independence in 1776.

Blockquotes

It is used to present content from another source:

With great powers come greater responsibilities

By Spiderman

Multiple Code Lines

For multiple lines of code, use the pre element:

Codingal is on a
mission to
inspire all school
children

Keyboard Inputs

To indicate input that is typically entered via the keyboard, use the kbd element:

Use ctrl + pto open the print dialog box.
